Datasets

FaceForensics++

The "FaceForensics++" dataset provides real and manipulated videos to train and evaluate deepfake detection techniques. With a focus on manipulated facial images, this dataset aids researchers in improving detection models using Convolutional Neural Networks (CNNs). It includes diverse manipulations such as Deepfakes, Face2Face, FaceSwap, and NeuralTextures, making it a valuable resource for combating digital forgery.

Deepfake Detection Challenge Dataset (DFDC)

The DFDC dataset is a collaborative effort to advance deepfake detection technology. It contains a diverse set of real and manipulated videos using techniques like face swapping and facial reenactment. This dataset supports robust model training, helping address challenges like video compression and manipulation diversity.

Celeb-DF

The Celeb-DF dataset features a large-scale collection of celebrity videos with diverse conditions. It challenges detection algorithms with real and manipulated videos generated by advanced techniques. This dataset is pivotal for developing robust models that ensure media authenticity and integrity.
