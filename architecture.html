Model Architecture

Our model combines the ResNext CNN for frame-level feature extraction and an LSTM network for temporal sequence processing. Pre-trained ResNext50_32x4d extracts 2048-dimensional feature vectors, which are sequentially analyzed by the LSTM to classify videos as deepfake or pristine. Dropout, Leaky ReLU, and adaptive pooling enhance model robustness. A SoftMax layer is used for prediction confidence.

ResNext

ResNext is a high-performance residual CNN optimized for deeper networks. We utilize the ResNext50_32x4d model to extract meaningful feature vectors, fine-tuned with additional layers and learning rates for gradient convergence. These vectors form the input for sequential processing.

LSTM for Temporal Analysis

The LSTM layer processes sequential frames to capture temporal dependencies, using a 2048-dimensional hidden layer with dropout to improve generalization. Sequential analysis ensures robust detection by comparing frames over time.
